How BSG Works?

The Basketball Statistics Generator is a progressive web application that records the statistics of each player in a team during the game. The statistics in the game would be tallied, computed, and sorted. Coaches are given a copy of the output and can use these statistics to analyze the strengths and weaknesses of their team to help the coach discuss the next strategy to be attempted for the team and in their next following games.

The Basketball Statistics Generator lets the user input the statistics of each player in every team which are the following:

  • two points - the user would input two points made by the player
  • three points- the user would input three points made by the player
  • Missed - the user would input if the player missed the shot
  • Assist - the user would input the player who was assisted by the shotM
  • Free throw - the user would input the number of times the player would take a free throw
  • Free throw missed - the user would input wether the player missed the shot in a free throw
  • Rebound - the user would input the number of times the player has rebound the attempted shot
  • Turn over - the user would input the number of turn overs in the player
